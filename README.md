\# Climate Temperature Analysis in Mexico (1902–2011)

\## Overview
This project analyzes long-term temperature trends in Mexico using climatic datasets.  
It includes data processing, visualizations, and a professional climate report.

\## Objectives
\- Identify regions with the greatest warming trends.
\- Analyze temperature extremes.
\- Calculate thermal amplitude.
\- Support real-world decision-making in agriculture, energy, urban planning, and climate policy.

\## Folder Structure
\- \*\*data/raw/\*\* – original climate files  
\- \*\*data/interim/\*\* – intermediate geospatial files  
\- \*\*data/processed/\*\* – final CSV dataset  
\- \*\*src/\*\* – scripts for processing and visualization  
\- \*\*reports/\*\* – climate report and figures  
\- \*\*powerbi/\*\* – dashboard Power BI file
